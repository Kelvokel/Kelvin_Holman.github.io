Warehouse AI

Personal Project
Python · FastAPI · SQLAlchemy · PostgreSQL/DuckDB · JavaScript · OpenCV · REST APIs

A full-stack warehouse operations platform built to mirror real receiving, inventory, and shipping workflows — including the kind of exception-handling and approval routing that real warehouse floors actually need.

  • Layered backend (routers → services → repositories → models) built on FastAPI and SQLAlchemy, with a schema modeled across 10+ relational entities and designed for PostgreSQL migration.
  • STOP_WAIT exception handling that automatically routes workflow mismatches to role-based supervisor approval queues, instead of letting bad movements reach fulfillment.
  • Mobile-friendly camera verification (barcode/QR + image matching) tested across iPhone/iPad browsers, with server-side scan fallback.
  • Audit logging on every transaction for traceability, plus unit and workflow tests covering core business logic.
